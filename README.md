# 🛩️ Расчёт минимальных безопасных высот (QFE / QNH)

Приложение для расчёта безопасных высот полёта с учётом рельефа местности и температуры воздуха.

---

## 📋 Что вам понадобится

У вас уже установлен **Docker** — этого достаточно!

---

## 🚀 Как запустить (пошагово)

### Шаг 1: Откройте терминал

- **Windows:** Нажмите `Win + R`, введите `cmd` и нажмите Enter


### Шаг 2: Перейдите в папку с проектом

В терминале введите команду (замените путь на ваш):

```bash
cd /путь/к/папке/вадим_проект
```



### Шаг 3: Соберите Docker-образ

```bash
docker build -t safe-height .
```

⏱️ Займёт 2–5 минут при первом запуске (скачиваются базовые компоненты).

### Шаг 4: Запустите приложение

```bash
docker run -p 8501:8501 safe-height
```

### Шаг 5: Откройте приложение в браузере

Перейдите по ссылке: **http://localhost:8501**

---

## 📖 Как пользоваться приложением

1. **Задайте параметры аэродрома** в левой панели:
   - Высота аэродрома (м)
   - Температура воздуха (°C)
   - Минимальный запас высоты (330 или 660 футов)
   - Угол оси ВПП (градусы)
   - Максимальная высота на карте (м)

2. **Загрузите карту высот** — изображение в формате PNG или JPG (градиент от чёрного к белому, где белый = макс. высота)

3. **Нажмите кнопку** «🚀 Открыть карту в интерактивном окне»

4. **В открывшемся окне:**
   - Кликните по точке порога ВПП (базовая точка)
   - Кликните по конечной точке оси ВПП
   - Нажмите клавишу **`s`** для сохранения (или **`q`** для отмены)

5. **Получите результат** — минимальная безопасная высота в метрах и футах

## 🛑 Как остановить приложение

1. Вернитесь в терминал, где запущено приложение
2. Нажмите **`Ctrl + C`** (Windows/Linux) или **`Cmd + C`** (macOS)

Или просто закройте окно терминала.

---

## 📁 Какие файлы нужны для работы

- `app.py` — основной файл приложения
- `calc.py` — формулы расчёта
- `cv2_selector.py` — инструмент выбора точек на карте
- `requirements.txt` — список библиотек
- `Dockerfile` — инструкция для Docker

## ❓ Проблемы и решения

| Проблема | Решение |
|----------|---------|
| Порт 8501 уже занят | Замените в команде запуска: `-p 8502:8501` и откройте `http://localhost:8502` |
| Docker не запускается | Убедитесь, что Docker Desktop запущен (иконка кита в трее) |
| Ошибка при сборке | Проверьте, что все файлы на месте и есть интернет |
